---
title: 'Get Script Loader'
openapi: 'GET /loader/{serverId}/{scriptId}'
---

## Description

Fetch the script loader for a specific server and script.  
This endpoint returns a loader script that users can run in Roblox to start using your authenticated script.

## Path Parameters

| Name       | Type   | Description                         |
|------------|--------|-------------------------------------|
| serverId   | string | Discord server ID                   |
| scriptId   | string | Unique ID of the script             |

## Response

Returns the loader script for the specified server and script.
