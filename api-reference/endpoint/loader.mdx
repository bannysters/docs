## Description

Retrieve the script loader for a specific Discord server and script.  
This endpoint returns a Lua script that users can run in Roblox to begin using your authenticated script.

## Path Parameters

| Name     | Type   | Description               |
|----------|--------|---------------------------|
| serverId | string | Discord server ID         |
| scriptId | string | Unique identifier of the script |

## Example Request

To retrieve the loader, send a `GET` request to the following URL:

**Endpoint Format**  
```
GET https://9auth.xyz/loader/{serverId}/{scriptId}  
```

<CodeGroup>

```lua LuaU
local HttpService = game:GetService("HttpService")

local server_id = "1234567890"
local script_id = "ABC1234script"

local httpRequest =
    (syn and syn.request) or
    (SENTINEL_V2 and function(opt)
        return {
            StatusCode = 200,
            Body = request(opt.Url, opt.Method, opt.Body or "")
        }
    end) or
    request or
    function()
        return { Body = "{}" }
    end

local response = httpRequest({
    Url = "https://9auth.xyz/loader/" .. server_id .. "/" .. script_id,
    Method = "GET",
    Headers = {
      ["Content-Type"] = "application/json"
    }
})

print(response.text)
```

```javascript Node JS
const axios = require('axios');
const server_id = "1234567890";
const script_id = "ABC1234script";

axios.get(`https://9auth.xyz/loader/${server_id}/${script_id}`)
  .then(response => {
    console.log(response.text);
  })
  .catch(error => {
    console.error('Error:', error.response.data);
  });
```
```python Python
import requests

server_id = "1234567890"
script_id = "ABC1234script"

url = "https://9auth.xyz/loader/" + server_id + "/" + script_id

response = requests.get(url)
print(response.text)
```

</CodeGroup>

Returns a Lua script loader for the specified server and script.
